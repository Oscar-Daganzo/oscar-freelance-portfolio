Proyectos

Experiencia real en producto

EPE – Españoles Por Europa

Red social para españoles que viven, trabajan o viajan por Europa. Conecta personas por ciudad y país, pensada para facilitar su integración y encontrar contactos en cada destino.

  • Mapa con clustering inteligente por ciudad (optimizado para 50.000+ usuarios).
  • Perfiles personalizables, temas visuales y sistema de logros/medallas.
  • Chat en tiempo real tipo WhatsApp: paginación, scroll inteligente, estados de lectura.
  • Notificaciones push, bloqueo de usuarios y fondos personalizados por chat.
React Native Expo Firebase · Firestore FCM Push

Woodem – Herramienta para carpintería

Aplicación profesional orientada a talleres de carpintería y fabricación de muebles. A partir de las medidas del mueble, genera automáticamente el despiece, el cálculo de materiales y la merma.

  • Algoritmo de despiece y cálculo de tableros optimizado para uso diario.
  • Formulario avanzado de entrada de datos con opciones de baldas, puertas y divisores.
  • Generación de PDF con cortes y piezas listo para producción.
TypeScript Expo Algoritmos de cálculo
